Monologues Next
===============

Synopsis
--------

Prototype of updated Shakespeare's Monologues site

Devlopment
----------

 - Install ruby 2.2.0
 - git clone
 - bundle install
 - rake test
 - make changes
 - rake test
 - git push
 - travis ci runs automatically
 - heroku deploys automatically if ci build passes
